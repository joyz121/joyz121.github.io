# 二叉树(BinaryTree)

二叉树的每个结点至多有两棵子树。

## 一、二叉树实现

通过之前学习的链式结构实现二叉树。每个树结点包括3个部分：

- 数据域
- 左指针域：指向左子树
- 右指针域：指向右子树

使用结构体定义树结点：

```c++
typedef struct BinaryTreeNode
{
    double data;
    BinaryTreeNode* Lchild,*Rchild;
}BinTreeNode;
```

接下来手动生成一个二叉树

```c++
//插入子树
BinTreeNode* Insert(double Data){
	BinTreeNode* BT;
	BT=(BinTreeNode*)malloc(sizeof(BinTreeNode));
	BT->data=Data;
	BT->Lchild=NULL;
	BT->Rchild=NULL;
	return BT;
}
//创建树
BinTreeNode* CreatBinTree()
{
    BinTreeNode* BT;
    BT=(BinTreeNode*)malloc(sizeof(BinTreeNode));
    BT->data=1;
    BT->Lchild=Insert(2);
    BT->Rchild=Insert(3);
    BT->Lchild->Rchild=Insert(4);
    BT->Lchild->Rchild->Rchild=Insert(5);
    return BT;
}
```

## 二、二叉树遍历——递归

### 2.1 先序递归遍历

根——左——右

```c++
void RePreOrderTraversal(BinTreeNode *BT)
{
    if(BT)
    {
        cout<<BT->data;
        RePreOrderTraversal(BT->Lchild);
        RePreOrderTraversal(BT->Rchild);
    }
}
```

### 2.2 中序递归遍历

左——根——右

```c++
void ReInOrderTraversal(BinTreeNode* BT)
{
    if(BT)
    {
        ReInOrderTraversal(BT->Lchild);
        cout<<BT->data;
        ReInOrderTraversal(BT->Rchild);
    }
}
```

### 2.3 后序递归遍历

左——右——根

```c++
void RePostOrderTraversal(BinTreeNode* BT)
{
    if(BT)
    {
        RePostOrderTraversal(BT->Lchild);
        RePostOrderTraversal(BT->Rchild);
        cout<<BT->data;
    }
}
```

## 三、二叉树遍历——非递归

### 3.1 先序非递归遍历

```c++
void PreOrderTraversal(BinTreeNode *BT)
{
    BinTreeNode* p=BT;
    BinTreeNode* q;
    Stack* S=InitStack();//创建链栈
    while(p||!IsEmpty(S))//树或栈非空
    {
        if(p)
        {
            cout<<p->data;//打印根结点
            Push(S,p);//根结点入栈
            p=p->Lchild;//进入左子树
        }
        else
        {
            q=Pop(S);//根结点出栈
            p=q->Rchild;
        }
    }
}
```

### 后序非递归遍历

```c++
void PostOrderTraversal(BinTreeNode* BT)
{
    BinTreeNode* p=BT;
    BinTreeNode* q;
    Stack* S=InitStack();
    while(p||!IsEmpty(S))
    {
        if(p)
        {
            Push(S,p);//根结点入栈
            p=p->Lchild;//进入左子树
        }
        else//左子树空
        {
            p=Pop(S);//p指向栈顶
            Push(S,p);//p重新入栈
            if(p->Rchild&&(p->Rchild!=q))//右子树存在且未入
            {
                p=p->Rchild;
            }
            else
            {
                q=Pop(S);//记录上次访问的根结点
                cout<<q->data;
                p=nullptr;
            }
        }
    }
}
```
